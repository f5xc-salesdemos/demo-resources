CDN 模拟器是一台运行 NGINX 的 Azure VM，配置为模拟 CDN 边缘节点的行为。它注入真实的厂商特定标头，并模拟缓存行为，用于与 F5 Distributed Cloud 的 CDN 集成演示。

## 模拟厂商

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

有关 nginx 配置、厂商标头、terraform 变量及部署详情，请参阅 [CDN 模拟器文档](https://f5-sales-demo.github.io/cdn-simulator/)。
