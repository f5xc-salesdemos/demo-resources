流量生成器是一台 Azure 虚拟机，用于针对 F5 Distributed Cloud HTTP 负载均衡器生成攻击流量、侦察扫描和机器人模拟。它可以验证 WAF 策略、Bot Defense、API Security 和 Client-Side Defense 是否已正确配置。

## 验证的 F5 XC 功能

- **Web 应用防火墙** — SQL 注入、XSS、命令注入、路径遍历载荷
- **Bot Defense** — 无头浏览器自动化、凭证填充、快速爬取
- **API Security** — OWASP API Top 10 攻击、GraphQL 滥用、参数模糊测试
- **Client-Side Defense** — Magecart 风格的窃取脚本注入、DOM 操纵

请参阅 [流量生成器文档](https://f5xc-salesdemos.github.io/traffic-generator/) 了解工具目录、流量套件、terraform 变量和部署详情。
