流量生成器是一台 Azure VM，用于针对 F5 Distributed Cloud HTTP 负载均衡器产生攻击流量、侦察扫描和机器人模拟。它可验证 WAF 策略、Bot 防护、API 安全以及客户端防护是否已正确配置。

## 已验证的 F5 XC 功能

- **Web 应用防火墙** — SQL 注入、XSS、命令注入、路径遍历载荷
- **Bot 防护** — 无头浏览器自动化、凭据填充、快速爬取
- **API 安全** — OWASP API Top 10 攻击、GraphQL 滥用、参数模糊测试
- **客户端防护** — Magecart 风格的信息窃取脚本注入、DOM 操纵

请参阅[流量生成器文档](https://f5-sales-demo.github.io/traffic-generator/)，了解工具目录、流量套件、Terraform 变量及部署详情。
