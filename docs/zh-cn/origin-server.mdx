源服务器是一台 Azure VM，通过 nginx 反向代理托管易受攻击的 Web 应用程序。它为演示 F5 Distributed Cloud 安全功能提供测试目标。

## 已验证的 F5 XC 功能

- **Web 应用防火墙** — SQL 注入、XSS、命令注入、路径遍历
- **API 安全** — REST 和 GraphQL API 漏洞、OWASP API Top 10
- **机器人防御** — 凭证填充目标、爬取端点
- **客户端防御** — JavaScript 注入、Magecart 式攻击

有关架构、应用程序、Terraform 变量和部署详情，请参阅[源服务器文档](https://f5xc-salesdemos.github.io/origin-server/)。
