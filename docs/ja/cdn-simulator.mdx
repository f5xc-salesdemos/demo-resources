CDN シミュレーターは、CDN エッジノードとして動作するよう設定された NGINX を実行する Azure VM です。リアルなベンダー固有のヘッダーを注入し、F5 Distributed Cloud との CDN インテグレーションデモ向けにキャッシュ動作をシミュレートします。

## シミュレート対象ベンダー

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

NGINX 設定、ベンダーヘッダー、Terraform 変数、およびデプロイの詳細については、[CDN シミュレーターのドキュメント](https://f5-sales-demo.github.io/cdn-simulator/)を参照してください。
