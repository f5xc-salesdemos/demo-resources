CDN シミュレーターは、CDN エッジノードのように動作するよう設定された NGINX を実行する Azure VM です。リアルなベンダー固有のヘッダーを挿入し、F5 Distributed Cloud との CDN 統合デモ用にキャッシュ動作をシミュレートします。

## シミュレート対象ベンダー

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

nginx 設定、ベンダーヘッダー、terraform 変数、デプロイの詳細については、[CDN シミュレーターのドキュメント](https://f5xc-salesdemos.github.io/cdn-simulator/)を参照してください。
