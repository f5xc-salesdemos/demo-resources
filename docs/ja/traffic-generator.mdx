トラフィックジェネレーターは、F5 Distributed Cloud HTTP ロードバランサーに対して攻撃トラフィック、偵察スキャン、およびボットシミュレーションを生成する Azure VM です。WAF ポリシー、Bot 高度防御、API セキュリティ、およびクライアントサイド防御が正しく設定されていることを検証します。

## 検証される F5 XC 機能

- **Web アプリファイアウォール (WAF)** — SQL インジェクション、XSS、コマンドインジェクション、パストラバーサルのペイロード
- **Bot 高度防御** — ヘッドレスブラウザ自動化、クレデンシャルスタッフィング、高速クローリング
- **API セキュリティ** — OWASP API Top 10 攻撃、GraphQL 悪用、パラメーターファジング
- **クライアントサイド防御** — Magecart スタイルのスキマーインジェクション、DOM 操作

ツールカタログ、トラフィックスイート、Terraform 変数、およびデプロイの詳細については、[トラフィックジェネレーターのドキュメント](https://f5xc-salesdemos.github.io/traffic-generator/)を参照してください。
