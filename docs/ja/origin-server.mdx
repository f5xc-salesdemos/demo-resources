オリジンサーバーは、nginx リバースプロキシの背後に脆弱な Web アプリケーションをホストする Azure VM です。F5 Distributed Cloud のセキュリティ機能のデモ用ターゲットを提供します。

## 検証済み F5 XC 機能

- **Web アプリファイアウォール (WAF)** — SQL インジェクション、XSS、コマンドインジェクション、パストラバーサル
- **API セキュリティ** — REST および GraphQL API の脆弱性、OWASP API Top 10
- **Bot 防御** — クレデンシャルスタッフィングターゲット、スクレイピングエンドポイント
- **クライアントサイド防御** — JavaScript インジェクション、Magecart スタイルの攻撃

アーキテクチャ、アプリケーション、Terraform 変数、およびデプロイの詳細については、[オリジンサーバーのドキュメント](https://f5-sales-demo.github.io/origin-server/)を参照してください。
