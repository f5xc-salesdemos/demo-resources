オリジンサーバーは、nginx リバースプロキシの背後で脆弱な Web アプリケーションをホストする Azure VM です。F5 Distributed Cloud のセキュリティ機能をデモンストレーションするためのターゲットを提供します。

## 検証される F5 XC 機能

- **Web アプリケーションファイアウォール** — SQLインジェクション、XSS、コマンドインジェクション、パストラバーサル
- **API セキュリティ** — REST および GraphQL API の脆弱性、OWASP API Top 10
- **ボット防御** — クレデンシャルスタッフィングのターゲット、スクレイピングエンドポイント
- **クライアントサイド防御** — JavaScriptインジェクション、Magecartスタイルの攻撃

アーキテクチャ、アプリケーション、Terraform 変数、およびデプロイの詳細については、[オリジンサーバーのドキュメント](https://f5xc-salesdemos.github.io/origin-server/)を参照してください。
