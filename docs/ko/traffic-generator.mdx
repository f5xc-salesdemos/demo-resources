트래픽 생성기는 F5 Distributed Cloud HTTP 로드 밸런서를 대상으로 공격 트래픽, 정찰
스캔 및 봇 시뮬레이션을 생성하는 Azure VM입니다. 이를 통해
WAF 정책, Bot Defense, API Security 및 Client-Side Defense가
올바르게 구성되었는지 검증합니다.

## 검증되는 F5 XC 기능

- **Web Application Firewall** — SQL 인젝션, XSS, 커맨드 인젝션, 경로 탐색 페이로드
- **Bot Defense** — 헤드리스 브라우저 자동화, 크리덴셜 스터핑, 고속 크롤링
- **API Security** — OWASP API Top 10 공격, GraphQL 남용, 파라미터 퍼징
- **Client-Side Defense** — Magecart 스타일 스키머 삽입, DOM 조작

도구 카탈로그, 트래픽 스위트, terraform 변수 및 배포 세부 사항에 대해서는 [트래픽 생성기 문서](https://f5xc-salesdemos.github.io/traffic-generator/)를
참조하십시오.
