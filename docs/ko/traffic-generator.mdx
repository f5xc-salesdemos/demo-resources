트래픽 생성기는 F5 Distributed Cloud HTTP 부하 분산 장치에 대해 공격 트래픽, 정찰
스캔 및 봇 시뮬레이션을 생성하는 Azure VM입니다. 이를 통해 웹 앱 방화벽 (WAF) 정책, Bot 고급 방어, API 보안, 클라이언트 측 방어가
올바르게 구성되었는지 검증합니다.

## 검증되는 F5 XC 기능

- **웹 애플리케이션 방화벽** — SQL 인젝션, XSS, 명령 인젝션, 경로 탐색 페이로드
- **Bot 고급 방어** — 헤드리스 브라우저 자동화, 크리덴셜 스터핑, 급속 크롤링
- **API 보안** — OWASP API Top 10 공격, GraphQL 악용, 파라미터 퍼징
- **클라이언트 측 방어** — Magecart 스타일 스키머 인젝션, DOM 조작

도구 카탈로그, 트래픽 스위트, Terraform 변수 및 배포 세부 정보는 [트래픽 생성기 문서](https://f5-sales-demo.github.io/traffic-generator/)를
참조하십시오.
