CDN 시뮬레이터는 CDN 엣지 노드처럼 동작하도록 구성된 NGINX를 실행하는 Azure VM입니다.
현실적인 벤더별 헤더를 주입하고, F5 Distributed Cloud와의 CDN 통합 데모를 위한
캐시 동작을 시뮬레이션합니다.

## 시뮬레이션된 벤더

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

nginx 구성, 벤더 헤더, Terraform 변수 및 배포 세부 정보는 [CDN 시뮬레이터 문서](https://f5xc-salesdemos.github.io/cdn-simulator/)를 참조하십시오.
