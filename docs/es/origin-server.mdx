El Servidor de origen es una máquina virtual de Azure que aloja aplicaciones web vulnerables detrás
de un proxy inverso nginx. Proporciona objetivos para demostrar las funciones de seguridad
de F5 Distributed Cloud.

## Funciones de F5 XC validadas

- **Firewall de aplicaciones web (WAF)** — inyección SQL, XSS, inyección de comandos, traversal de rutas
- **Seguridad de API** — vulnerabilidades de API REST y GraphQL, OWASP API Top 10
- **Defensa Bot** — objetivos de relleno de credenciales, endpoints de scraping
- **Defensa del lado del cliente** — inyección de JavaScript, ataques estilo Magecart

Consulte la [documentación del Servidor de origen](https://f5-sales-demo.github.io/origin-server/)
para conocer la arquitectura, las aplicaciones, las variables de Terraform y los detalles de implementación.
