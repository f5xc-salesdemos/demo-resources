El simulador CDN es una VM de Azure que ejecuta NGINX configurado para comportarse como un
nodo de borde CDN. Inyecta encabezados realistas específicos de cada proveedor y simula el comportamiento
de caché para demostraciones de integración CDN con F5 Distributed Cloud.

## Proveedores Simulados

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consulte la [documentación del CDN Simulator](https://f5xc-salesdemos.github.io/cdn-simulator/)
para la configuración de nginx, encabezados de proveedores, variables de terraform y detalles de despliegue.
