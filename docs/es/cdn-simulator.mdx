El Simulador CDN es una VM de Azure que ejecuta NGINX configurado para comportarse como un
nodo de borde CDN. Inyecta cabeceras realistas específicas de cada proveedor y simula el
comportamiento de caché para demostraciones de integración CDN con F5 Distributed Cloud.

## Proveedores simulados

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consulte la [documentación del Simulador CDN](https://f5-sales-demo.github.io/cdn-simulator/)
para la configuración de nginx, cabeceras de proveedor, variables de terraform y detalles de despliegue.
