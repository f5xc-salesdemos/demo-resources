El generador de tráfico es una VM de Azure que produce tráfico de ataque, escaneos
de reconocimiento y simulación de bots contra balanceadores de carga HTTP de F5 Distributed Cloud. Valida que las políticas de Firewall de aplicaciones web (WAF), Defensa Bot, Seguridad de API y Defensa del lado del cliente estén correctamente configuradas.

## Funcionalidades de F5 XC validadas

- **Firewall de aplicaciones web** — inyección SQL, XSS, inyección de comandos, cargas útiles de path traversal
- **Defensa Bot** — automatización de navegadores sin interfaz gráfica, relleno de credenciales, rastreo acelerado
- **Seguridad de API** — ataques OWASP API Top 10, abuso de GraphQL, fuzzing de parámetros
- **Defensa del lado del cliente** — inyección de skimmer estilo Magecart, manipulación del DOM

Consulte la [documentación del Generador de tráfico](https://f5-sales-demo.github.io/traffic-generator/)
para el catálogo de herramientas, suites de tráfico, variables de Terraform y detalles de implementación.
