來源伺服器是一台 Azure VM，透過 nginx 反向代理託管易受攻擊的網頁應用程式。它提供用於展示 F5 Distributed Cloud 安全功能的測試目標。

## 已驗證的 F5 XC 功能

- **網頁應用程式防火牆** — SQL 注入、XSS、命令注入、路徑遍歷
- **API 安全性** — REST 與 GraphQL API 漏洞、OWASP API Top 10
- **機器人防禦** — 憑證填充目標、爬取端點
- **用戶端防禦** — JavaScript 注入、Magecart 類型攻擊

請參閱[來源伺服器文件](https://f5-sales-demo.github.io/origin-server/)，以了解架構、應用程式、Terraform 變數及部署詳細資訊。
