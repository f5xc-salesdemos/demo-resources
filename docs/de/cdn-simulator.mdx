Der CDN-Simulator ist eine Azure-VM, auf der NGINX ausgeführt wird und die so konfiguriert ist, dass sie sich wie ein CDN-Edge-Knoten verhält. Er injiziert realistische herstellerspezifische Header und simuliert Cache-Verhalten für CDN-Integrationsdemos mit F5 Distributed Cloud.

## Simulierte Anbieter

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Weitere Informationen zur NGINX-Konfiguration, zu Vendor-Headern, Terraform-Variablen und Deployment-Details finden Sie in der [CDN-Simulator-Dokumentation](https://f5xc-salesdemos.github.io/cdn-simulator/).
