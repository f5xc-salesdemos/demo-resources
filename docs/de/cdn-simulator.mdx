Der CDN-Simulator ist eine Azure-VM, auf der NGINX läuft und so konfiguriert ist, dass sie sich wie ein CDN-Edge-Knoten verhält. Er injiziert realistische anbieterspezifische Header und simuliert das Cache-Verhalten für CDN-Integrationsdemos mit F5 Distributed Cloud.

## Simulierte Anbieter

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Weitere Informationen zur NGINX-Konfiguration, zu Anbieter-Headern, Terraform-Variablen und Deployment-Details finden Sie in der [CDN-Simulator-Dokumentation](https://f5-sales-demo.github.io/cdn-simulator/).
