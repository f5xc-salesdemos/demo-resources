Der Ursprungsserver ist eine Azure VM, die anfällige Webanwendungen hinter
einem nginx-Reverse-Proxy hostet. Er stellt Ziele für die Demonstration von
F5 Distributed Cloud-Sicherheitsfunktionen bereit.

## Validierte F5 XC-Funktionen

- **Web-App-Firewall (WAF)** — SQL-Injection, XSS, Befehlsinjektion, Pfadüberquerung
- **API-Sicherheit** — REST- und GraphQL-API-Schwachstellen, OWASP API Top 10
- **Bot-Abwehr** — Credential-Stuffing-Ziele, Scraping-Endpunkte
- **Clientseitige Abwehr** — JavaScript-Injection, Magecart-ähnliche Angriffe

Siehe die [Ursprungsserver-Dokumentation](https://f5-sales-demo.github.io/origin-server/)
für Architektur, Anwendungen, Terraform-Variablen und Bereitstellungsdetails.
