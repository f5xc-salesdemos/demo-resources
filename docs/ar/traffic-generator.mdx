مولّد حركة المرور هو جهاز Azure افتراضي (VM) ينتج حركة مرور هجومية وعمليات
فحص استطلاعية ومحاكاة بوتات ضد موازنات تحميل HTTP في F5 Distributed Cloud. يتحقق
من أن سياسات WAF وBot Defense وAPI Security وClient-Side Defense
مُكوّنة بشكل صحيح.

## ميزات F5 XC التي يتم التحقق منها

- **جدار حماية تطبيقات الويب** — حقن SQL، البرمجة عبر المواقع (XSS)، حقن الأوامر، حمولات اجتياز المسار
- **الحماية من البوتات** — أتمتة المتصفح بدون واجهة، حشو بيانات الاعتماد، الزحف السريع
- **أمان واجهات برمجة التطبيقات** — هجمات OWASP API Top 10، إساءة استخدام GraphQL، التشويش على المعاملات
- **الحماية من جانب العميل** — حقن أدوات سرقة البيانات بأسلوب Magecart، التلاعب بـ DOM

راجع [توثيق مولّد حركة المرور](https://f5xc-salesdemos.github.io/traffic-generator/)
للاطلاع على كتالوج الأدوات وحزم حركة المرور ومتغيرات terraform وتفاصيل النشر.
