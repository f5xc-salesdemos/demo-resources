مولد حركة المرور عبارة عن جهاز Azure VM يُنتج حركة مرور هجومية وعمليات مسح استطلاعية ومحاكاة للبوت ضد موازنات تحميل HTTP في F5 Distributed Cloud. يتحقق هذا المولد من صحة تهيئة سياسات جدار حماية تطبيقات الويب (WAF) ودفاع Bot وأمان API والدفاع من جهة العميل.

## ميزات F5 XC التي يتم التحقق منها

- **جدار حماية تطبيقات الويب (WAF)** — حقن SQL وXSS وحقن الأوامر وحمولات اجتياز المسار
- **دفاع Bot** — أتمتة المتصفحات بلا واجهة رسومية وحشو بيانات الاعتماد والزحف السريع
- **أمان API** — هجمات OWASP API Top 10 وإساءة استخدام GraphQL وضبابية المعاملات
- **الدفاع من جهة العميل** — حقن برامج التجسس بأسلوب Magecart ومعالجة DOM
