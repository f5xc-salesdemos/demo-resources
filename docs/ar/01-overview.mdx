موارد العرض التوضيحي هي أجهزة افتراضية Azure مُهيأة مسبقاً وقابلة للنشر عبر Terraform.
كل مكوّن عبارة عن جهاز افتراضي مستقل مثبّت عليه برنامج مخصص ومُهيأ
من خلال cloud-init عند الإقلاع الأول.

## اختيار المكونات

المكونات هي لبنات البناء. اختَرها بناءً على ما يحتاجه عرضك التوضيحي:

- **خادم المصدر** يوفر تطبيقات ويب قابلة للاختراق تُشكّل أهدافاً لميزات الأمان في F5 XC. معظم العروض التوضيحية تحتاج إليه.
- **مولد حركة المرور** ينتج حركة مرور هجومية وحركة مرور مشروعة تجاه بيئة العرض التوضيحي. ضروري عند إظهار فاعلية جدار حماية تطبيقات الويب (WAF)، أو اكتشاف البوت، أو حماية API.
- **محاكي CDN** يضع حافة CDN مبنية على NGINX بين العملاء والمصدر، مع حقن رؤوس خاصة بالموردين. يُستخدم لعروض تكامل CDN التوضيحية.

## المتطلبات الأساسية المشتركة

تتطلب جميع المكونات:

- اشتراك Azure مع صلاحية المساهم
- Terraform 1.5 أو أحدث
- واجهة سطر أوامر Azure مُصادَق عليها (`az login`)
- زوج مفاتيح SSH (المسار الافتراضي `~/.ssh/id_ed25519.pub`)

## نمط النشر

كل مكوّن يتبع نفس سير عمل Terraform:

1. استنسِخ مستودع المكوّن
2. `cd terraform/`
3. انسخ `terraform.tfvars.example` إلى `terraform.tfvars` وعيّن المتغيرات المطلوبة (`subscription_id` مطلوب للجميع؛ وبعض المكونات لها مدخلات مطلوبة إضافية)
4. `terraform init && terraform plan && terraform apply`
5. تحقق عبر نقطة نهاية فحص الصحة الخاصة بالمكوّن

يتم حل معرّف المنشئ تلقائياً من حساب Azure AD الخاص بك ويُضمَّن في جميع أسماء الموارد (مثل: `rg-origin-server-lab-rmordasiewicz`).

## التكامل بين المكونات

تتحد المكونات لإنشاء بيئات عرض توضيحي متكاملة:

```
Traffic Generator ──► F5 XC HTTP LB (WAF/Bot/API/CSD) ──► Origin Server
                                     ▲
                               CDN Simulator
```

**ترتيب النشر:**

1. **خادم المصدر** — انشره أولاً، إذ يستضيف التطبيقات التي تستهدفها جميع المكونات الأخرى
2. **موازن تحميل HTTP لـ F5 XC** — هيّئ مجموعة مصدر تشير إلى العنوان IP العام لخادم المصدر، ثم أنشئ موازن تحميل HTTP مع سياسات جدار حماية تطبيقات الويب (WAF)، أو دفاع Bot، أو أمان API، أو الدفاع من جهة العميل
3. **مولد حركة المرور** و/أو **محاكي CDN** — انشرهما أخيراً، مشيرَين إلى FQDN موازن تحميل F5 XC

تُشير متغيرات Terraform الخاصة بكل مكوّن إلى مخرجات المكونات المنشورة مسبقاً. راجع وثائق المكوّن للاطلاع على متطلبات المتغيرات المحددة.

استخدم [حاسبة أسعار Azure](https://azure.microsoft.com/en-us/pricing/calculator/) للحصول على تقديرات التكلفة الحالية. شغّل `terraform destroy` عندما يكون المختبر خاملاً لإيقاف الرسوم.
