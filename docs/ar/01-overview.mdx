موارد العرض التوضيحي هي أجهزة Azure افتراضية مُعدَّة مسبقاً وقابلة للنشر عبر Terraform.
كل مكون هو جهاز افتراضي مستقل مع برمجيات مُصمَّمة لغرض محدد، يتم تثبيتها وتهيئتها
من خلال cloud-init عند أول تشغيل.

## اختيار المكونات

المكونات هي وحدات بناء. اختر منها بناءً على ما يحتاجه عرضك التوضيحي:

- **خادم الأصل** يوفر تطبيقات ويب معرَّضة للثغرات تعمل كأهداف لميزات أمان
  F5 XC. معظم العروض التوضيحية تحتاج إلى هذا المكون.
- **مولِّد حركة المرور** ينتج حركة مرور هجومية وشرعية ضد بيئة العرض التوضيحي
  الخاصة بك. مطلوب عند عرض فعالية WAF أو اكتشاف الروبوتات أو حماية واجهات API.
- **محاكي CDN** يضع حافة CDN مبنية على NGINX بين العملاء والأصل،
  مع حقن ترويسات خاصة بالمورِّد. يُستخدم لعروض تكامل CDN التوضيحية.

## المتطلبات المسبقة المشتركة

تتطلب جميع المكونات:

- اشتراك Azure مع صلاحية مساهم
- Terraform 1.5 أو أحدث
- Azure CLI مُصادَق عليه (`az login`)
- زوج مفاتيح SSH (المسار الافتراضي `~/.ssh/id_ed25519.pub`)

## نمط النشر

يتبع كل مكون نفس سير عمل Terraform:

1. استنساخ مستودع المكون
2. `cd terraform/`
3. نسخ `terraform.tfvars.example` إلى `terraform.tfvars` وتعيين المتغيرات المطلوبة (`subscription_id` مطلوب لجميع المكونات؛ بعض المكونات لديها مدخلات مطلوبة إضافية)
4. `terraform init && terraform plan && terraform apply`
5. التحقق عبر نقطة فحص صحة المكون

يتم حل معرِّف الناشر تلقائياً من حساب Azure AD الخاص بك ويُضمَّن في جميع أسماء الموارد (مثال: `rg-origin-server-lab-rmordasiewicz`).

## التكامل بين المكونات

تتجمع المكونات لإنشاء بيئات عرض توضيحي متكاملة:

```
Traffic Generator ──► F5 XC HTTP LB (WAF/Bot/API/CSD) ──► Origin Server
                                     ▲
                               CDN Simulator
```

**ترتيب النشر:**

1. **خادم الأصل** — يُنشر أولاً، فهو يستضيف التطبيقات التي تستهدفها جميع المكونات الأخرى
2. **موازن أحمال HTTP الخاص بـ F5 XC** — قم بتهيئة مجموعة أصل تشير إلى عنوان IP العام لخادم الأصل، ثم أنشئ موازن أحمال HTTP مع سياسات WAF أو Bot Defense أو API Security أو Client-Side Defense
3. **مولِّد حركة المرور** و/أو **محاكي CDN** — يُنشران أخيراً، مع توجيههما إلى اسم النطاق الكامل (FQDN) لموازن أحمال F5 XC

تُشير متغيرات Terraform لكل مكون إلى مخرجات المكونات المنشورة مسبقاً. راجع وثائق المكون لمعرفة متطلبات المتغيرات المحددة.

استخدم [حاسبة أسعار Azure](https://azure.microsoft.com/en-us/pricing/calculator/) للحصول على تقديرات التكلفة الحالية. قم بتشغيل `terraform destroy` عندما يكون المختبر خاملاً لإيقاف الرسوم.
