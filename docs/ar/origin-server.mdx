خادم الأصل هو جهاز Azure VM يستضيف تطبيقات ويب معرضة للاختراق خلف
وكيل عكسي nginx. يوفر أهدافًا لعرض ميزات أمان F5 Distributed
Cloud.

## ميزات F5 XC التي تم التحقق منها

- **جدار حماية تطبيقات الويب** — حقن SQL، XSS، حقن الأوامر، اجتياز المسار
- **أمان API** — ثغرات REST وGraphQL API، قائمة OWASP API العشر الأوائل
- **الحماية من الروبوتات** — أهداف حشو بيانات الاعتماد، نقاط نهاية الاستخراج
- **الحماية من جانب العميل** — حقن JavaScript، هجمات بأسلوب Magecart

راجع [وثائق خادم الأصل](https://f5xc-salesdemos.github.io/origin-server/)
للاطلاع على البنية المعمارية والتطبيقات ومتغيرات terraform وتفاصيل النشر.
