خادم المصدر هو جهاز Azure VM يستضيف تطبيقات ويب قابلة للاختراق خلف
وكيل nginx عكسي. يوفر أهدافًا لتوضيح ميزات الأمان في F5 Distributed
Cloud.

## ميزات F5 XC التي تم التحقق منها

- **جدار حماية تطبيقات الويب (WAF)** — حقن SQL وXSS وحقن الأوامر واجتياز المسار
- **أمان API** — ثغرات REST وGraphQL API، وأعلى 10 ثغرات في OWASP API
- **دفاع Bot** — أهداف حشو بيانات الاعتماد ونقاط نهاية الاستخراج
- **الدفاع من جهة العميل** — حقن JavaScript وهجمات بأسلوب Magecart

راجع [وثائق خادم المصدر](https://f5-sales-demo.github.io/origin-server/)
للاطلاع على التفاصيل المتعلقة بالبنية والتطبيقات ومتغيرات Terraform وتفاصيل النشر.
