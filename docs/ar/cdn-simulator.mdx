محاكي CDN هو جهاز Azure VM يعمل بتشغيل NGINX مُهيَّأ للتصرف كعقدة حافة CDN.
يقوم بحقن رؤوس واقعية خاصة بالموردين ويحاكي سلوك التخزين المؤقت للعروض التوضيحية لتكامل CDN مع F5 Distributed Cloud.

## الموردون المُحاكَون

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

راجع [توثيق محاكي CDN](https://f5-sales-demo.github.io/cdn-simulator/)
للاطلاع على تهيئة nginx، ورؤوس الموردين، ومتغيرات Terraform، وتفاصيل النشر.
