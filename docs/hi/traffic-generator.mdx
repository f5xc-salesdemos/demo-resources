ट्रैफ़िक जनरेटर एक Azure VM है जो F5 Distributed Cloud HTTP लोड बैलेंसर के विरुद्ध अटैक ट्रैफ़िक, रिकॉनेसेंस स्कैन और बॉट सिमुलेशन उत्पन्न करता है। यह सत्यापित करता है कि WAF नीतियाँ, Bot Defense, API सुरक्षा और क्लाइंट-साइड डिफेंस सही तरीके से कॉन्फ़िगर की गई हैं।

## सत्यापित F5 XC सुविधाएँ

- **वेब ऐप फ़ायरवॉल (WAF)** — SQL इंजेक्शन, XSS, कमांड इंजेक्शन, पाथ ट्रैवर्सल पेलोड
- **Bot उन्नत रक्षा** — हेडलेस ब्राउज़र ऑटोमेशन, क्रेडेंशियल स्टफिंग, रैपिड क्रॉलिंग
- **API सुरक्षा** — OWASP API Top 10 अटैक, GraphQL दुरुपयोग, पैरामीटर फ़ज़िंग
- **क्लाइंट-साइड डिफेंस** — Magecart-स्टाइल स्किमर इंजेक्शन, DOM मैनिपुलेशन

उपकरण कैटलॉग, ट्रैफ़िक सुइट्स, Terraform वेरिएबल और डिप्लॉयमेंट विवरण के लिए [ट्रैफ़िक जनरेटर दस्तावेज़ीकरण](https://f5-sales-demo.github.io/traffic-generator/) देखें।
