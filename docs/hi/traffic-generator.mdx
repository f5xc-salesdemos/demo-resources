ट्रैफ़िक जनरेटर एक Azure VM है जो F5 Distributed Cloud HTTP लोड बैलेंसरों के विरुद्ध
अटैक ट्रैफ़िक, रिकॉनसेंस स्कैन और बॉट सिमुलेशन उत्पन्न करता है। यह सत्यापित करता है
कि WAF नीतियाँ, Bot Defense, API Security और Client-Side Defense सही ढंग से
कॉन्फ़िगर किए गए हैं।

## F5 XC द्वारा सत्यापित फ़ीचर्स

- **Web Application Firewall** — SQL इंजेक्शन, XSS, कमांड इंजेक्शन, पाथ ट्रैवर्सल पेलोड
- **Bot Defense** — हेडलेस ब्राउज़र ऑटोमेशन, क्रेडेंशियल स्टफ़िंग, रैपिड क्रॉलिंग
- **API Security** — OWASP API Top 10 अटैक, GraphQL दुरुपयोग, पैरामीटर फ़ज़िंग
- **Client-Side Defense** — Magecart-शैली स्किमर इंजेक्शन, DOM मैनिपुलेशन

टूल कैटलॉग, ट्रैफ़िक सुइट्स, terraform वेरिएबल्स और डिप्लॉयमेंट विवरण के लिए [Traffic Generator दस्तावेज़ीकरण](https://f5xc-salesdemos.github.io/traffic-generator/) देखें।
