CDN सिम्युलेटर एक Azure VM है जिसमें NGINX कॉन्फ़िगर किया गया है ताकि यह CDN
एज नोड की तरह व्यवहार करे। यह यथार्थवादी वेंडर-विशिष्ट हेडर इंजेक्ट करता है और F5 Distributed Cloud के साथ CDN एकीकरण डेमो के लिए कैश
व्यवहार का अनुकरण करता है।

## अनुकरित वेंडर

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door
