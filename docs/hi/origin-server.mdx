ऑरिजिन सर्वर एक Azure VM है जो nginx रिवर्स प्रॉक्सी के पीछे कमज़ोर वेब एप्लिकेशन
होस्ट करता है। यह F5 Distributed Cloud सुरक्षा सुविधाओं का प्रदर्शन करने के लिए
लक्ष्य प्रदान करता है।

## सत्यापित F5 XC सुविधाएँ

- **वेब ऐप फ़ायरवॉल (WAF)** — SQL इंजेक्शन, XSS, कमांड इंजेक्शन, पाथ ट्रैवर्सल
- **API सुरक्षा** — REST और GraphQL API कमज़ोरियाँ, OWASP API Top 10
- **Bot रक्षा** — क्रेडेंशियल स्टफिंग लक्ष्य, स्क्रेपिंग एंडपॉइंट
- **क्लाइंट-साइड डिफेंस** — JavaScript इंजेक्शन, Magecart-शैली के हमले

आर्किटेक्चर, एप्लिकेशन, terraform वेरिएबल, और डिप्लॉयमेंट विवरण के लिए
[ऑरिजिन सर्वर दस्तावेज़ीकरण](https://f5xc-salesdemos.github.io/origin-server/) देखें।
