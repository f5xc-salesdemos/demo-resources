Le serveur d'origine est une machine virtuelle Azure qui héberge des applications
web vulnérables derrière un proxy inverse nginx. Il fournit des cibles pour la
démonstration des fonctionnalités de sécurité de F5 Distributed Cloud.

## Fonctionnalités F5 XC validées

- **Pare-feu applicatif (WAF)** — injection SQL, XSS, injection de commandes, traversée de répertoires
- **Sécurité des API** — vulnérabilités des API REST et GraphQL, OWASP API Top 10
- **Défense Bot** — cibles de credential stuffing, points de terminaison de scraping
- **Défense côté client** — injection JavaScript, attaques de type Magecart

Consultez la [documentation du Serveur d'origine](https://f5-sales-demo.github.io/origin-server/)
pour l'architecture, les applications, les variables Terraform et les détails de déploiement.
