Le Simulateur CDN est une VM Azure exécutant NGINX configuré pour se comporter comme un
nœud de périphérie CDN. Il injecte des en-têtes spécifiques à chaque fournisseur et simule le
comportement du cache pour les démonstrations d'intégration CDN avec F5 Distributed Cloud.

## Fournisseurs simulés

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consultez la [documentation du Simulateur CDN](https://f5xc-salesdemos.github.io/cdn-simulator/)
pour la configuration nginx, les en-têtes des fournisseurs, les variables Terraform et les détails de déploiement.
