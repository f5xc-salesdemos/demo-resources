Le générateur de trafic est une machine virtuelle Azure qui produit du trafic d'attaque, des
analyses de reconnaissance et des simulations de bots contre les équilibreurs de charge HTTP F5 Distributed Cloud. Il
valide que les politiques WAF, la Défense Bot, la Sécurité des API et la Défense côté client
sont correctement configurées.

## Fonctionnalités F5 XC validées

- **Pare-feu applicatif (WAF)** — injections SQL, XSS, injections de commandes, charges utiles de traversée de chemin
- **Défense Bot** — automatisation de navigateur sans interface, credential stuffing, exploration intensive
- **Sécurité des API** — attaques OWASP API Top 10, abus GraphQL, fuzzing de paramètres
- **Défense côté client** — injection de skimmer de type Magecart, manipulation du DOM

Consultez la [documentation du Générateur de trafic](https://f5xc-salesdemos.github.io/traffic-generator/)
pour le catalogue d'outils, les suites de trafic, les variables Terraform et les détails de déploiement.
