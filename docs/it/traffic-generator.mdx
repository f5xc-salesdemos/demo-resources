Il generatore di traffico è una VM Azure che produce traffico di attacco, scansioni
di ricognizione e simulazione bot contro i load balancer HTTP di F5 Distributed Cloud. Consente di
verificare che le policy WAF, la Difesa Bot, la Sicurezza API e la Difesa lato client
siano configurate correttamente.

## Funzionalità F5 XC Validate

- **Firewall per applicazioni web (WAF)** — payload di SQL injection, XSS, command injection e path traversal
- **Difesa Bot** — automazione con browser headless, credential stuffing, crawling rapido
- **Sicurezza API** — attacchi OWASP API Top 10, abuso GraphQL, parameter fuzzing
- **Difesa lato client** — iniezione di skimmer in stile Magecart, manipolazione del DOM

Consultare la [documentazione del Generatore di traffico](https://f5xc-salesdemos.github.io/traffic-generator/)
per il catalogo degli strumenti, le suite di traffico, le variabili Terraform e i dettagli di distribuzione.
