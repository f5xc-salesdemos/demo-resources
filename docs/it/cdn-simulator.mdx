Il simulatore CDN è una VM Azure che esegue NGINX configurata per comportarsi come un
nodo edge CDN. Inietta header specifici per vendor realistici e simula il comportamento
della cache per le demo di integrazione CDN con F5 Distributed Cloud.

## Vendor Simulati

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consulta la [documentazione del Simulatore CDN](https://f5xc-salesdemos.github.io/cdn-simulator/)
per la configurazione nginx, gli header vendor, le variabili terraform e i dettagli di distribuzione.
