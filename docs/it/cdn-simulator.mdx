Il CDN simulator è una VM Azure che esegue NGINX configurato per comportarsi come un
nodo edge CDN. Inietta header realistici specifici del vendor e simula il comportamento
della cache per demo di integrazione CDN con F5 Distributed Cloud.

## Vendor simulati

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consultare la [documentazione del CDN Simulator](https://f5xc-salesdemos.github.io/cdn-simulator/)
per la configurazione di nginx, gli header dei vendor, le variabili terraform e i dettagli di deployment.
