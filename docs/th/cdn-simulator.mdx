CDN simulator คือ Azure VM ที่รัน NGINX ซึ่งถูกกำหนดค่าให้ทำงานเสมือนเป็น CDN
edge node โดยจะแทรก header เฉพาะของผู้ให้บริการแต่ละรายอย่างสมจริง และจำลองพฤติกรรม cache
สำหรับการสาธิตการผสานรวม CDN กับ F5 Distributed Cloud

## ผู้ให้บริการที่จำลอง

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

ดู[เอกสารประกอบ CDN Simulator](https://f5xc-salesdemos.github.io/cdn-simulator/)
สำหรับการกำหนดค่า nginx, header ของผู้ให้บริการ, ตัวแปร terraform และรายละเอียดการ deploy
