ตัวจำลอง CDN คือ Azure VM ที่รัน NGINX ซึ่งได้รับการกำหนดค่าให้ทำงานเหมือน CDN
edge node โดยจะฉีด header ที่เฉพาะเจาะจงสำหรับผู้ให้บริการแต่ละรายอย่างสมจริง และจำลองพฤติกรรมแคชสำหรับการสาธิตการผสานรวม CDN กับ F5 Distributed Cloud

## ผู้ให้บริการที่รองรับการจำลอง

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

ดู[เอกสารตัวจำลอง CDN](https://f5-sales-demo.github.io/cdn-simulator/)
สำหรับรายละเอียดการกำหนดค่า nginx, header ของผู้ให้บริการ, ตัวแปร Terraform และรายละเอียดการปรับใช้งาน
