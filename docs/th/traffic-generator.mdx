ตัวสร้างทราฟฟิกคือ Azure VM ที่ผลิตทราฟฟิกโจมตี การสแกนสอดแนม
และการจำลอง Bot เพื่อทดสอบกับ F5 Distributed Cloud HTTP load balancer โดยใช้ในการ
ตรวจสอบว่านโยบาย WAF, การป้องกัน Bot, ความปลอดภัย API และการป้องกันฝั่งไคลเอนต์
ได้รับการกำหนดค่าอย่างถูกต้อง

## ฟีเจอร์ F5 XC ที่ได้รับการตรวจสอบ

- **ไฟร์วอลล์แอปเว็บ (WAF)** — SQL injection, XSS, command injection, เพย์โหลด path traversal
- **การป้องกัน Bot** — การทำงานอัตโนมัติด้วย headless browser, credential stuffing, การ crawling อย่างรวดเร็ว
- **ความปลอดภัย API** — การโจมตี OWASP API Top 10, การละเมิด GraphQL, parameter fuzzing
- **การป้องกันฝั่งไคลเอนต์** — การฉีด skimmer แบบ Magecart, การจัดการ DOM
