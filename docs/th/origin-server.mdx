Origin server เป็น Azure VM ที่โฮสต์เว็บแอปพลิเคชันที่มีช่องโหว่ไว้เบื้องหลัง
nginx reverse proxy โดยเป็นเป้าหมายสำหรับการสาธิตฟีเจอร์ด้านความปลอดภัยของ F5 Distributed
Cloud

## ฟีเจอร์ F5 XC ที่ผ่านการตรวจสอบ

- **Web Application Firewall** — SQL injection, XSS, command injection, path traversal
- **API Security** — ช่องโหว่ของ REST และ GraphQL API, OWASP API Top 10
- **Bot Defense** — เป้าหมายการโจมตี credential stuffing, scraping endpoints
- **Client-Side Defense** — JavaScript injection, การโจมตีรูปแบบ Magecart
