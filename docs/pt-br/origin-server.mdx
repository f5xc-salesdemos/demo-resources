O servidor de origem é uma VM Azure que hospeda aplicações web vulneráveis por trás
de um proxy reverso nginx. Ele fornece alvos para demonstrar os recursos de segurança
do F5 Distributed Cloud.

## Recursos F5 XC Validados

- **Web Application Firewall** — injeção de SQL, XSS, injeção de comandos, path traversal
- **Segurança de API** — vulnerabilidades em APIs REST e GraphQL, OWASP API Top 10
- **Defesa contra Bots** — alvos de credential stuffing, endpoints de scraping
- **Defesa do Lado do Cliente** — injeção de JavaScript, ataques estilo Magecart

Consulte a [documentação do Servidor de Origem](https://f5-sales-demo.github.io/origin-server/)
para arquitetura, aplicações, variáveis terraform e detalhes de implantação.
