O simulador de CDN é uma VM Azure executando NGINX configurado para se comportar como um nó de borda CDN. Ele injeta cabeçalhos específicos de fornecedores realistas e simula o comportamento de cache para demonstrações de integração CDN com o F5 Distributed Cloud.

## Fornecedores Simulados

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consulte a [documentação do Simulador de CDN](https://f5-sales-demo.github.io/cdn-simulator/)
para configuração do nginx, cabeçalhos de fornecedores, variáveis do Terraform e detalhes de implantação.
